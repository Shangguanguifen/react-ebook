react-ebook是一个使用 JavaScript(JS) 开发的简易的移动端电子阅读器。我使用 react、redux、react-router 搭建了项目的基本框架，引入epubjs实现了阅读器的各种常用的组件。借助React hooks、Redux hooks创建了各种function组件替换class 组件实现应用功能
